DataFooter
==========

.. aknowledgement::
    This page is a Python adaptation of the `official Vuetify documentation <https://v2.vuetifyjs.com/en/api/v-data-footer/>`_.
    All examples have been converted to ipyvuetify syntax.

The :py:class:`DataFooter <ipyvuetify.DataFooter>` component is used internally by :py:class:`DataTable <ipyvuetify.DataTable>` and :py:class:`DataIterator <ipyvuetify.DataIterator>` to render the pagination footer. It displays pagination controls including items-per-page selector and page navigation.

.. note::
    This is a sub-component that is automatically rendered by :py:class:`DataTable <ipyvuetify.DataTable>` and :py:class:`DataIterator <ipyvuetify.DataIterator>`. You can customize its behavior using the ``footer_props`` prop on the parent component.
