DataTableHeader
===============

.. aknowledgement::
    This page is a Python adaptation of the `official Vuetify documentation <https://v2.vuetifyjs.com/en/api/v-data-table-header/>`_.
    All examples have been converted to ipyvuetify syntax.

The :py:class:`DataTableHeader <ipyvuetify.DataTableHeader>` component is used internally by :py:class:`DataTable <ipyvuetify.DataTable>` to render the table header with column titles and sort controls.

.. note::
    This is a sub-component that is automatically rendered by :py:class:`DataTable <ipyvuetify.DataTable>`. You typically don't need to use this component directly, but instead configure headers through the ``headers`` prop on the parent :py:class:`DataTable <ipyvuetify.DataTable>` component.
