Data
====

.. aknowledgement::
    This page is a Python adaptation of the `official Vuetify documentation <https://v2.vuetifyjs.com/en/api/v-data/>`_.
    All examples have been converted to ipyvuetify syntax.

The :py:class:`Data <ipyvuetify.Data>` component is an abstract base component that provides shared functionality for data display components like :py:class:`DataTable <ipyvuetify.DataTable>` and :py:class:`DataIterator <ipyvuetify.DataIterator>`. It handles common data operations including sorting, filtering, and pagination.

.. note::
    This is an abstract component and is not meant to be used directly. Use :py:class:`DataTable <ipyvuetify.DataTable>` or :py:class:`DataIterator <ipyvuetify.DataIterator>` instead.

.. seealso::

    * :py:class:`ipyvuetify.DataTable` - Display data in table format
    * :py:class:`ipyvuetify.DataIterator` - Display data with custom layouts

.. api::

    :py:class:`ipyvuetify.Data`
