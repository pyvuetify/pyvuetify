The ``CardActions`` component is a sub-component of :py:class:`Card <ipyvuetify.Card>` that provides a container for action buttons within a card. It automatically handles the layout and spacing of buttons, typically positioned at the bottom of the card.

.. note::
    This component is typically used within :py:class:`Card <ipyvuetify.Card>`. See the :doc:`Card documentation <Card>` for usage examples.
