ipyvuetify.SlideItem
====================

.. py:class:: ipyvuetify.SlideItem

   Bases: :py:obj:`ipyvuetify.generated.VuetifyWidget.VuetifyWidget`


   A single item within a v-slide-group component.


   .. py:attribute:: active_class

      Configure the active CSS class applied when the link is active. You can find more information about the [**active-class** prop](https://router.vuejs.org/api/#active-class) on the vue-router documentation.



   .. py:attribute:: disabled

      Removes the ability to click or target the component.



   .. py:attribute:: value

      Controls whether the component is visible or hidden.
