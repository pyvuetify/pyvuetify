ipyvuetify.Stepper
==================

.. py:class:: ipyvuetify.Stepper

   Bases: :py:obj:`ipyvuetify.generated.VuetifyWidget.VuetifyWidget`


   The v-stepper component displays progress through numbered steps, commonly used for multi-step processes.


   .. py:attribute:: alt_labels

      Places the labels beneath the step



   .. py:attribute:: dark

      Applies the dark theme variant to the component. You can find more information on the Material Design documentation for [dark themes](https://material.io/design/color/dark-theme.html).



   .. py:attribute:: light

      Applies the light theme variant to the component.



   .. py:attribute:: non_linear

      Allow user to jump to any step



   .. py:attribute:: value

      Controls whether the component is visible or hidden.



   .. py:attribute:: vertical

      Display steps vertically
